# FlickerProgressBar

### 自定义属性

| 属性名 | 说明 | 默认值 |
|--------|--------|--------|
|     loadingColor   |    下载中颜色    | 	#40c4ff 	|
|     stopColor   |    暂停时和下载完成颜色    | 		#ff9800 	|
|     textSize   |    进度文本字体大小    | 	12sp		|
|     radius   |    圆角    | 		0dp	|
|     borderWidth   |    背景边框宽度    | 		1dp 	|
